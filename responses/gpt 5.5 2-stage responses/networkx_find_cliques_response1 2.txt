1. Every yielded clique contains only nodes from `G`, with no repeated nodes within a clique.

2. Every yielded clique is complete: for every pair of distinct nodes in the clique, an edge exists between them in `G`.

3. Every yielded clique is maximal: no node outside the clique can be added while preserving completeness.

4. No maximal clique is yielded more than once, ignoring the order of nodes within each clique.

5. If `nodes` is provided and is itself a clique, every yielded clique contains all nodes in `nodes`, and only maximal cliques satisfying that constraint are yielded.
