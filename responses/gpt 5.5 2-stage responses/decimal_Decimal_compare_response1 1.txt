2. If either operand is `NaN`, the result is `Decimal('NaN')`.

3. If neither operand is `NaN`, the result is always one of `Decimal('-1')`, `Decimal('0')`, or `Decimal('1')`.

4. If neither operand is `NaN`, the result is `Decimal('0')` if and only if the two operands are numerically equal.

5. If neither operand is `NaN`, comparing in the opposite order negates the result: `a.compare(b) == -b.compare(a)`.
