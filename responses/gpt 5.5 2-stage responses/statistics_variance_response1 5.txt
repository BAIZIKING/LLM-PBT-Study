1. For any valid data set with at least two values, the returned variance is never negative.

2. When `xbar` is omitted or is the true mean of the data, the result equals the sum of squared deviations from the mean divided by `n - 1`.

3. When `xbar` is omitted or is the true mean, the variance is zero if and only if all data values are equal.

4. Adding the same constant to every data value does not change the variance.

5. Multiplying every data value by a constant `k` multiplies the variance by `k²`.
