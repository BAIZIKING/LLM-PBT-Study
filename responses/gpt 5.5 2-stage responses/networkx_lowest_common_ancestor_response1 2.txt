1. The output is symmetric in the two query nodes: `lowest_common_ancestor(G, node1, node2)` should equal `lowest_common_ancestor(G, node2, node1)`.

2. If `node1 == node2`, the output should be `node1`.

3. If the output is not `default`, it should be a node in `G` that is an ancestor of both `node1` and `node2`, allowing a node to be its own ancestor.

4. If there is no common ancestor of `node1` and `node2`, the output should be exactly `default`.

5. If the output is not `default`, it should be “lowest”: there should be no other common ancestor of `node1` and `node2` that is a proper descendant of the returned node.
