1. Every yielded clique is a list whose elements are nodes from `G`.

2. Every yielded clique is complete: for every pair of distinct nodes in the clique, an edge exists between them in `G`.

3. Every yielded clique is maximal: no node outside the clique can be added while preserving completeness.

4. The output contains no duplicate cliques when cliques are compared as sets, regardless of node order within each clique.

5. If `nodes` is provided and is itself a clique, every yielded maximal clique contains all nodes in `nodes`; if `nodes` is not a clique, the function raises `ValueError`.
