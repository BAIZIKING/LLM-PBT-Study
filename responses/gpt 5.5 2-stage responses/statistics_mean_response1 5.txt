1. For any non-empty numeric data, the result is equal to the sum of the data divided by the number of data points.

2. The result is independent of the order of the input values; permuting the data does not change the mean.

3. If all input values are the same value `x`, the result is exactly `x`.

4. For non-empty real-valued data, the result lies between the minimum and maximum input values, inclusive.

5. Adding a constant `c` to every input value increases the result by `c`, and multiplying every input value by a constant `k` multiplies the result by `k`.
