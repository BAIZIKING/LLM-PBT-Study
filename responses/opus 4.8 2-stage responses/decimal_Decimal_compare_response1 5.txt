2. If neither operand is a NaN, the result is exactly one of Decimal('-1'), Decimal('0'), or Decimal('1').

3. If either operand is a NaN, the result is a NaN (Decimal('NaN')).

4. The comparison is antisymmetric: for non-NaN operands, compare(a, b) equals the negation of compare(b, a) (i.e., swapping the arguments negates the result).

5. For non-NaN operands, the result equals Decimal('0') if and only if the two operands are numerically equal, Decimal('-1') if and only if the first operand is less than the second, and Decimal('1') if and only if the first operand is greater than the second.
