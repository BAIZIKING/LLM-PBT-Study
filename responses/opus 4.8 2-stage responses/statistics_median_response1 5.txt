1. The median is always within the range of the data, i.e., it is greater than or equal to the minimum value and less than or equal to the maximum value of the input.

2. The median is invariant under permutation of the input data, i.e., reordering or shuffling the elements does not change the result.

3. For input data with an odd number of elements, the median is an actual member of the input data (one of the original values).

4. For input data with an even number of elements, the median equals the average of the two middle values of the sorted data, and equals (sorted[n//2 - 1] + sorted[n//2]) / 2.

5. Adding a constant offset to every element of the input shifts the median by that same constant (i.e., median(x + c for all x) == median(data) + c), and scaling every element by a positive constant scales the median by that constant.
