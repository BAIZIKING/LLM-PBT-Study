1. The variance is always non-negative (output ≥ 0) for any valid input data.

2. The variance is zero if and only if all data values are identical (constant data yields exactly 0).

3. The variance is invariant under translation: adding a constant to every data point leaves the variance unchanged.

4. The variance scales by the square of a multiplicative factor: multiplying every data point by a constant c multiplies the variance by c².

5. Passing the correct mean of the data as the optional xbar argument produces the same result as omitting it (calling variance(data) equals variance(data, mean(data))).
