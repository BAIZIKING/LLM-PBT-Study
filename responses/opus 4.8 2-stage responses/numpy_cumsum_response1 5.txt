1. The output array has the same total number of elements as the input array `a` (the size is preserved).

2. The last element of the cumulative sum (over the flattened array, or along the specified axis) equals the total sum of the elements (along that same axis), up to floating-point rounding.

3. The first element of the output (along the cumulative axis) equals the first element of the input (along that axis).

4. The element-wise difference between consecutive entries of the output (along the cumulative axis) equals the corresponding original input elements (i.e., `diff(cumsum(a)) == a[1:]` along that axis), up to floating-point rounding.

5. When `dtype` is specified, the output array has that dtype; otherwise the output dtype matches the input dtype (except integer inputs of precision lower than the platform integer are promoted to the default platform integer).
