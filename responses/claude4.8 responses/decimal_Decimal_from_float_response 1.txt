## Generation Strategy

To generate a wide variety of inputs and edge cases for `Decimal.from_float(f)`, I will use a combination of strategies:

1. **Floats**: Use `st.floats()` which can generate normal floats, subnormals, very large/small magnitudes, zeros (positive and negative), and special values like `NaN`, `inf`, and `-inf` (enabled via `allow_nan=True` and `allow_infinity=True`).
2. **Integers**: Use `st.integers()` since the documentation states `from_float` accepts `int` instances too, including very large integers (Python ints are arbitrary precision).
3. **Combine** both with `st.one_of()` so the test exercises both accepted input types.

## Properties to Check

Based on the API documentation:

1. **Return type**: The result must be a `Decimal` instance.
2. **Special values mapping**:
   - `NaN` → `Decimal('NaN')` (check via `is_nan()`).
   - `inf` → `Decimal('Infinity')` and `-inf` → `Decimal('-Infinity')` (check via `is_infinite()` and sign).
3. **Exactness / round-trip for finite values**: The created `Decimal` must equal the exact value of the float. Converting the `Decimal` back to a `float` must reproduce the original float bit-for-bit (since `from_float` stores the exact representable value). For integers, the `Decimal` must equal the integer exactly.
4. **Sign preservation**: For finite values (including signed zero), the sign of the float should be preserved.

@given(st.data())
def test_decimal_Decimal_from_float(data):
    value = data.draw(
        st.one_of(
            st.floats(allow_nan=True, allow_infinity=True),
            st.integers(),
        )
    )

    result = Decimal.from_float(value)

    # Property 1: result is always a Decimal
    assert isinstance(result, Decimal)

    # Property 2: special values mapping (only applies to floats)
    if isinstance(value, float) and math.isnan(value):
        assert result.is_nan()
        return

    if isinstance(value, float) and math.isinf(value):
        assert result.is_infinite()
        # Property 4 (sign): sign must match
        assert (result < 0) == (value < 0)
        return

    # Property 3: exact round-trip / equality for finite values
    if isinstance(value, float):
        # Decimal stores the exact value of the float, so converting back
        # to float must reproduce the original bits exactly.
        assert float(result) == value
        # Property 4: sign preservation, including signed zero.
        assert math.copysign(1.0, float(result)) == math.copysign(1.0, value)
    else:
        # For integers, the Decimal must equal the integer exactly.
        assert result == Decimal(value)
        assert int(result) == value
