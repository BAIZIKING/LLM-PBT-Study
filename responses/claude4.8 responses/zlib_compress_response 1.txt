## Generation Strategy

To generate a wide variety of inputs and edge cases for `zlib.compress`, I'll focus on the three parameters:

1. **`data`**: Use `st.binary()` to generate random bytes objects. By setting `min_size=0`, we include the empty-bytes edge case, and large `max_size` values to cover bigger inputs that actually exercise compression.

2. **`level`**: Use `st.integers(min_value=-1, max_value=9)` to cover all valid compression levels: `-1` (default), `0` (no compression), `1` through `9` (best speed to best compression). This hits all documented edge cases.

3. **`wbits`**: The documentation defines three valid ranges. I'll use `st.sampled_from()` combined with a union of integer ranges to cover:
   - `+9` to `+15` (zlib header/trailer)
   - `-9` to `-15` (raw stream, no header)
   - `+25` to `+31` (gzip header/trailer)

## Properties to Check

Based on the API documentation:

1. **Return type**: The result is always a `bytes` object.
2. **Round-trip correctness**: Compressing then decompressing (with matching `wbits`) recovers the original data exactly — this is the core invariant of a compressor. We must pass the same `wbits` to `zlib.decompress` since it determines header format.
3. **No exceptions on valid input**: Valid parameter combinations (per the documented ranges) should not raise errors.

```python
from hypothesis import given, strategies as st
import zlib

# Summary: Generate random bytes for `data` (including empty), all valid
# compression `levels` (-1 through 9), and `wbits` sampled from the three
# documented valid ranges (+9..+15 zlib, -9..-15 raw, +25..+31 gzip).
# Verify the result is bytes and round-trips back to the original data
# when decompressed with the same wbits.
@given(st.data())
def test_zlib_compress(data):
    payload = data.draw(st.binary(min_size=0, max_size=2048))
    level = data.draw(st.integers(min_value=-1, max_value=9))
    wbits = data.draw(
        st.one_of(
            st.integers(min_value=9, max_value=15),    # zlib header/trailer
            st.integers(min_value=-15, max_value=-9),  # raw stream
            st.integers(min_value=25, max_value=31),   # gzip header/trailer
        )
    )

    compressed = zlib.compress(payload, level, wbits)

    # Property 1: result is always bytes
    assert isinstance(compressed, bytes)

    # Property 2: round-trip recovers original data exactly
    # (decompress must use the same wbits to interpret the header format)
    decompressed = zlib.decompress(compressed, wbits)
    assert decompressed == payload
# End program
```
